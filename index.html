About Biippo

  • Car maintenance application developed for Android phones
  • Focuses on the 8th generation Honda Civic
  • Provides instructions for some of the most basic yet essential maintenance
  • Custom designed logo, buttons, and icons made using adobe illustrator

Biippo's User Focus

  • Chose 8th generation Honda Civic as first model due to the simplicity of maintenance and the popularity of the car model in order to reach a large user base
  • Designed a UI with user-friendliness and ease-of-use in mind
    • Appealing, welcoming color scheme
    • Custom and friendly icon design
  • Provided access to two different forms of instructions to allow users to choose their preferred style of learning
    • Step-by-step text instructions with images
    • Video instructions

Biippo's Technical Details

  • Application developed using Java in Android Studio
  • Created classes "Category" and "Subcategory" for the various maintenance categories
    • "Category" would contain an ArrayList of "Subcategory" instances
  • Pages would go through a switch statement of item IDs upon button selection to determine which intent, or activity, to create and start
  • Classes inmplemented "Parcelable", a fast form of serialization with more control designed specifically for Android applications
    • Watned to be able to store the "Category" and "Subcategory" objects on disk
  • Configured the various page layouts using XML files
